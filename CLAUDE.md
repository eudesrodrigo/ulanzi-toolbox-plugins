# Ulanzi D200 Plugins

This repo contains plugins for the Ulanzi D200 stream deck.

## Reference Implementation

`com.ulanzi.terminal-commander.ulanziPlugin/` is a working, tested plugin. Use it as the source of patterns and conventions.

## Plugin Installation

Plugins install to `~/Library/Application Support/Ulanzi/UlanziDeck/Plugins/`. After copying, run `npm install` inside the plugin directory and restart Ulanzi Studio.
